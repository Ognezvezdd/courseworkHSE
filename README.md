# Курсовая работа ПИ 2 курс
## Платформа симуляции конкурентного взаимодействия AI-агентов (Competitive Multi-Agent Simulation Platform)
### Авторы: Капогузов М. Е, Дудкина А. Э.

Платформа симуляции конкурентного взаимодействия AI-
агентов 

Ядро написано на **C++**, логика агентов - на **Python**, 
управление - через **Telegram-бота**.

## Основная идея

- Запуск матчей между агентами с разными стратегиями  
- Выбор игры и агента пользователем  
- Виртуальные ставки (баллы) на исход матча  
- Просмотр хода игры и результатов через Telegram

## Архитектура `(in progress)`

- **core/** - симуляционное ядро (C++)  
- **python_bridge/** - связь C++ <=> Python  
- **agents/** - реализации и SDK для агентов (Python)  
- **visualization/** - Telegram-бот и визуализация  
- **docs/** - документация  
- **tests/** - тесты  
- **examples/** - примеры сценариев и агентов

## Технологии `(in progress)`

C++17, Python 3, pybind11, Telegram Bot API, CMake, Docker

## Цель

Дать простой способ наблюдать, как разные AI-агенты соревнуются в формальных играх, и позволить пользователю "болеть" за выбранного агента с помощью виртуальных ставок
